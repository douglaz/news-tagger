# Example Narrative Tag

This is an example tag definition that demonstrates the format expected by news-tagger.

## Characteristics

Posts matching this narrative typically:

1. Use specific framing language
2. Appeal to emotions rather than facts
3. Reference authority figures without evidence
4. Create urgency or fear

## Examples

- "Experts say we must act now or face catastrophe"
- "Studies show that [vague claim]"
- "Everyone agrees that..."

## Notes

This definition is for testing purposes. Real definitions should be specific, objective, and based on clearly identifiable linguistic patterns.
